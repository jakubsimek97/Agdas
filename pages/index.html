pyAgdas

Processing of gravimetry data

Introduction

This project is for processing and analyzing data from absolute gravimeters FG5 and FG5X by Micro g LaCoste.
The project is based on script Agdas in Matlab. Vojtěch Pálinkáš and Miloš Vaľko are authors of Agdas script.
pyAgdas contains some features which are not included in g software by Micro g. The project was created for releasing of these knowledges. pyAgdas is released as open-source, so everybody is welcome to contribute.
