Instalation

pyAgdas is developed in Python 3.9.2. If you are a Windows user you can download it here. Users of Ubuntu have Python default.

Requirements

In the project a few external libraries are used. Here is their summary:

imagesc==0.2.1
matplotlib==3.3.4
numpy==1.20.1
pandas==1.2.3
Pillow==9.1.1
PyQt5==5.15.6
scipy==1.7.1

These requirements are in "requirements.txt" file, this file is generated by pipreqs app tool.

In case you do not have some tools installed in your Python, you can follow these commands for installing them:

Windows:

python -m pip install [name of the tool]

Example: python -m pip install scipy

Linux - Ubuntu:

pip install [name of the tool]

Example: pip install scipy

Run app

You can download the app here. You can clone the repository or you can download ZIP.

For running the app you have to run "main.py" script.
